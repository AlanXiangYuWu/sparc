├── algorithms/                  # 算法实现
│   ├── mappo.py                # MAPPO训练框架
│   ├── rmha_agent.py           # RMHA智能体
│   └── baselines/              # 基线算法
├── utils/                       # 工具函数

├── test.py                      # 测试脚本
├── visualize.py                 # 可视化脚本
└── experiments/                 # 实验脚本




训练RMHA模型

python train.py --config config/train_config_stable.yaml --algo rmha --seed 42


train_config_stable.yaml 是设置好的一个参数，机器人设置的比较少，这样训练比较快，  其他几个文件都是不同复杂度的参数设置， 如果你先训练更复杂的场景，就 直接参数调用其他 配置文件就可以。


代码没有问题，要想提高模型精度，就需要多加大训练步骤，具体参数建配置文档里面的 这个参数

  total_timesteps: 10000000    # 总训练步数




对比实验的：
# MAPPO (无通信)
python train.py --config config/train_config_stable.yaml --algo mappo --seed 42

# MAPPO + 图通信 (无距离编码)
python train.py --config config/train_config_stable.yaml --algo mappo_gnn --seed 42


训练不同的模型，后面的参数不同。 



测试：

python test.py --checkpoint results/checkpoints/rmha_seed42/rmha_best.pth --config config/test_config_stable.yaml



消融：
# 完整流程（训练+测试+可视化）
python experiments/ablation_study.py --train --test --visualize

# 仅测试（需要已训练模型）
python experiments/ablation_study.py --test --visualize

# 仅可视化（需要已有测试结果）
python experiments/ablation_study.py --visualize
